QUICK START
-----------

1. Read NC-AI-001-README.pdf for overview
2. Open NC-AI-001-Complete-Template.pdf to review risks
3. Choose your format:
   - PDF: Print and reference
   - Word: Edit and customize
   - CSV: Import to Excel/Sheets
   - Markdown: Web integration

4. Import CSV into Excel (if using spreadsheet)
5. Follow Excel setup instructions
6. Start assessing your AI risks!

EXCEL USERS
