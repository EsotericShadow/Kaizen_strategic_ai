### 3. Enable GitHub Pages (Simple - Settings → Pages)

**Where**: Settings → Pages (left sidebar)

**Steps**:
1. Click "Settings" tab
2. Click "Pages" in left sidebar
3. Under "Source": Select "Deploy from a branch"
4. Branch: `main`
5. Folder: `/ (root)`
6. Click "Save"

**Result**: Your README becomes a website at:
`https://esotericshadow.github.io/nc-ai-templates`

## Troubleshooting

**If you can't find Settings**:
- Make sure you're logged in
- Make sure you're the owner (EsotericShadow account)
- Settings tab is at the top of the repository page

**If Pages doesn't work**:
- Wait 5 minutes after enabling
- Check that you selected "main" branch
- Make sure README.md exists (it does)

**If topics don't save**:
- Click outside the input box after typing each one
- Make sure you're on the main repository page (Code tab)
