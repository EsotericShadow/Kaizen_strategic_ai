## Drafting Guidelines

### White Papers
- **Tone**: Professional, authoritative, evidence-based
- **Style**: Academic but accessible
- **Format**: PDF + web HTML
- **Citations**: APA style
- **Visuals**: Charts, diagrams, infographics

### Articles
- **Tone**: Conversational, accessible, practical
- **Style**: Engaging, clear, action-oriented
- **Format**: Web + PDF (for submissions)
- **Citations**: Hyperlinks or inline references
- **Length**: 500-1200 words (publication-specific)

### Case Studies
- **Tone**: Practical, testimonial-focused
- **Style**: Visual, story-driven
- **Format**: Web + PDF
- **Visuals**: Infographics, quotes, metrics
- **Length**: 1-2 pages

### Annual Reports
- **Tone**: Comprehensive, data-driven, professional
- **Style**: Report format, executive summary
- **Format**: PDF + web HTML
- **Visuals**: Charts, maps, infographics
- **Length**: 10-15 pages
