# ISO 42001 Certification Checklist

## The Only Checklist You Need

Use this. Print it. Check it off as you go.

---

## PHASE 1: GET READY (Month 1)

**Week 1-2: Setup**
- [ ] Buy ISO 42001 standard (online, ~$400)
- [ ] Read the simple workflows doc
- [ ] Decide: Consultant or DIY?
- [ ] Budget approved: $5K-$10K for certification

**Week 3-4: Foundation**
- [ ] Write AI Policy (1 page, see template below)
- [ ] List all AI tools you use (Google Sheet works)
- [ ] Make file folder: "ISO_42001_Documentation"
- [ ] Put policies in there

**Done**: You have basics. Moving on.

---

## PHASE 2: DOCUMENT YOUR PROCESSES (Month 2-3)

**Risk Assessment**
- [ ] Walk through each AI tool
- [ ] Ask: "What could go wrong?"
- [ ] Write it down (risk register)
- [ ] Plan to prevent each risk

**Client Projects**
- [ ] Document your typical project flow
- [ ] Make it a simple checklist
- [ ] Use it on next 3 projects
- [ ] Refine based on what works

**Incidents**
- [ ] Create incident log (spreadsheet)
- [ ] Log any issues or concerns
- [ ] Note how you fixed them

**Done**: You have procedures. They're simple. Good.

---

## PHASE 3: ACTUALLY USE THEM (Month 4-5)

**Daily**
- [ ] Follow daily AI checklist
- [ ] Think before you use AI
- [ ] Document anything important

**Weekly**
- [ ] Quick scan: Any problems?
- [ ] Update logs if needed

**Monthly**
- [ ] Update AI Systems List
- [ ] Review incident log
- [ ] Note improvements
- [ ] Save one-page summary

**Done**: You're consistently using your system.

---

## PHASE 4: CHECK YOURSELF (Month 6)

**Internal Audit** (Do this yourself or hire someone)
- [ ] Review all your documents
- [ ] Check you're following procedures
- [ ] Note what's missing
- [ ] Fix the gaps
- [ ] Document the fixes

**Done**: You're ready for official audit.

---

## PHASE 5: GET CERTIFIED (Month 7-8)

**Choose Auditor**
- [ ] Call 3 certification bodies
- [ ] Compare prices and timelines
- [ ] Pick one, sign contract

**Stage 1 Audit** (Document review)
- [ ] Send them your documents
- [ ] Answer their questions
- [ ] Fix any issues they find

**Stage 2 Audit** (Watch you work)
- [ ] Show them your processes
- [ ] Demonstrate you follow them
- [ ] Fix any issues they find

**Get Certificate**
- [ ] Done!
- [ ] Celebrate
- [ ] Update marketing materials

**Done**: You're certified! ğŸ‰

- [ ] Monthly check-in (15 min)

  - Best differentiation
  - Most credibility

- [ ] **Option B**: Build system but hold on certification (3 months, $500)

  - Build foundation for later
  - Lower cost entry

  - No cost
  - No audit
  - Still responsible AI

**Any option is fine.** Choose what fits your business.
