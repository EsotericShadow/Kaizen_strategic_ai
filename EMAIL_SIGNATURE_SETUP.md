# Email Signature Setup Guide

## Professional Email Signature for Kaizen Strategic AI

### HTML Version (Gmail, Outlook, Apple Mail, etc.)

**Copy the HTML code from `EMAIL_SIGNATURE.html` starting from the comment `<!-- HTML Email Signature -->`**

#### For Gmail:
1. Open Gmail → Settings → General
2. Scroll to "Signature"
3. Click "Create new" or edit existing
4. Paste the HTML signature
5. Save changes

#### For Outlook:
1. File → Options → Mail → Signatures
2. Create new signature
3. Paste HTML code (Outlook supports HTML)
4. Set as default for new messages and replies/forwards

#### For Apple Mail:
1. Mail → Preferences → Signatures
2. Create new signature
3. Paste HTML code
4. Set as default

### Plain Text Version (Basic email clients)

**Use the plain text version from `EMAIL_SIGNATURE_PLAIN.txt`** for:
- Basic email clients that don't support HTML
- Professional plain-text requirements
- Maximum compatibility

### Design Notes

- **Color Scheme**: Matches brand colors (Deep Ink #1E1E1E, Bluish Green accent)
- **Typography**: System fonts for maximum compatibility
- **Style**: Professional, minimalist, aligned with brand identity
- **Mobile-Friendly**: Responsive design that works on all devices

### Customization

You can customize:
- Font sizes (currently 13px/15px for compatibility)
- Colors (match brand palette)
- Border color (currently #80D0C7 - bluish green)
- Add phone number if desired
- Add social media links if desired

### Best Practices

1. **Keep it simple** - Less is more in email signatures
2. **Mobile-friendly** - Test on mobile devices
3. **Professional** - Avoid excessive formatting or images
4. **Consistent** - Use same signature across all email addresses
5. **Accessible** - Plain text fallback for accessibility
